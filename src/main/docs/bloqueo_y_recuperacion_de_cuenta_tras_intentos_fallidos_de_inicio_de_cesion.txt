TAREA #383 - SISTEMA DE PROTECCIÓN CONTRA FUERZA BRUTA Y RECUPERACIÓN DE CUENTA
═══════════════════════════════════════════════════════════════════════════════

Integrante a cargo: Jonatan

OBJETIVO:
Implementar un mecanismo automatizado de seguridad que proteja el sistema de 
logística contra ataques de fuerza bruta. Si un usuario (sin importar su rol) 
falla su contraseña 5 veces consecutivas, su cuenta se bloquea preventivamente, 
se genera un código de seguridad de 6 dígitos con vencimiento de 10 minutos y 
se despacha asíncronamente un correo vía la API de Resend para permitir su 
recuperación en dos pasos.

3. CLASE CREADA: SeguridadService.java
═══════════════════════════════════════════════════════════════════════════════

Ubicación: src/main/java/com/logitrack/sistema_logistica/service/SeguridadService.java

Propósito:
Concentrar la lógica pura de negocio de la seguridad sin ensuciar los módulos de logística. Reutiliza la infraestructura de WebSockets y correo asíncrono del equipo.

Código:

Java
@Service
@RequiredArgsConstructor
@Slf4j
public class SeguridadService {

    private final UsuarioRepository usuarioRepository;
    private final NotificationService notificationService; // Interfaz genérica del equipo
    private final AlertaWebService alertaWebService;       // Alertas WebSocket del equipo

    @Transactional
    public void manejarIntentoFallido(String username) {
        Optional<Usuario> usuarioOpt = usuarioRepository.findByUsername(username);
        if (usuarioOpt.isEmpty()) return;
        
        Usuario usuario = usuarioOpt.get();
        if (Boolean.TRUE.equals(usuario.getBloqueado())) return;

        int nuevosIntentos = usuario.getIntentosFallidos() + 1;
        usuario.setIntentosFallidos(nuevosIntentos);

        if (nuevosIntentos >= 5) {
            usuario.setBloqueado(true);
            String codigo = generarCodigoAleatorio();
            usuario.setCodigoDesbloqueo(codigo);
            usuario.setVencimientoCodigo(LocalDateTime.now().plusMinutes(10));
            
            String asunto = "LogiTrack - Código de Seguridad para Desbloqueo";
            String mensaje = "Tu cuenta ha sido bloqueada tras 5 intentos fallidos.\n"
                           + "Para recuperar el acceso, ingresá el siguiente código de 6 dígitos:\n\n"
                           + "CÓDIGO: " + codigo + "\n\n"
                           + "Este código expirará en 10 minutos.";
                           
            // Parche anti-errores para entornos de desarrollo locales con usuarios sin '@'
            String correoDestino = usuario.getUsername();
            if (!correoDestino.contains("@")) {
                correoDestino = "logitrack.agro@gmail.com";
                log.info("El username no es un mail. Redirigiendo código a: {}", correoDestino);
            }
            
            notificationService.enviarNotificacion(correoDestino, asunto, mensaje);
            log.info("Cuenta bloqueada. Código enviado a: {}", correoDestino);

            // Alerta por WebSockets en tiempo real a los Supervisores conectados
            usuarioRepository.findByRol(RolUsuario.SUPERVISOR).forEach(supervisor -> 
                alertaWebService.crearYEnviarAlerta(
                    supervisor, 
                    "El usuario " + usuario.getUsername() + " fue bloqueado por múltiples intentos fallidos.", 
                    "SEGURIDAD"
                )
            );
        }
        usuarioRepository.save(usuario);
    }

    @Transactional
    public void resetearIntentos(String username) {
        usuarioRepository.findByUsername(username).ifPresent(usuario -> {
            usuario.setIntentosFallidos(0);
            usuario.setUltimoAcceso(LocalDateTime.now());
            usuarioRepository.save(usuario);
        });
    }

    @Transactional
    public boolean verificarCodigoDesbloqueo(String username, String codigoIngresado) {
        Optional<Usuario> usuarioOpt = usuarioRepository.findByUsername(username);
        if (usuarioOpt.isPresent()) {
            Usuario usuario = usuarioOpt.get();
            if (Boolean.TRUE.equals(usuario.getBloqueado()) &&
                codigoIngresado.equals(usuario.getCodigoDesbloqueo()) &&
                usuario.getVencimientoCodigo() != null &&
                usuario.getVencimientoCodigo().isAfter(LocalDateTime.now())) {
                
                usuario.setBloqueado(false);
                usuario.setIntentosFallidos(0);
                usuario.setCodigoDesbloqueo(null);
                usuario.setVencimientoCodigo(null);
                usuarioRepository.save(usuario);
                return true;
            }
        }
        return false;
    }

    private String generarCodigoAleatorio() {
        Random random = new Random();
        return String.valueOf(100000 + random.nextInt(900000));
    }
}

    @PostMapping("/login")
    public ResponseEntity<?> login(@RequestBody LoginRequestDTO request) {
        Usuario usuario = usuarioRepository.findByUsername(request.getUsername()).orElse(null);

        if (usuario == null || !Boolean.TRUE.equals(usuario.getActivo())) {
            return ResponseEntity.status(HttpStatus.UNAUTHORIZED).body(Map.of("error", "Usuario no encontrado o inactivo"));
        }

        // Validación estricta preventiva de estado bloqueado
        if (Boolean.TRUE.equals(usuario.getBloqueado())) {
            return ResponseEntity.status(HttpStatus.FORBIDDEN)
                    .body(Map.of("error", "Cuenta bloqueada por seguridad. Revisa tu correo electrónico para ingresar el código de recuperación."));
        }

        if (!passwordEncoder.matches(request.getPassword(), usuario.getPasswordHash())) {
            seguridadService.manejarIntentoFallido(usuario.getUsername());
            
            // Solución al desfasaje de memoria: Forzar lectura fresca de la BD
            usuario = usuarioRepository.findByUsername(usuario.getUsername()).get();

            if (Boolean.TRUE.equals(usuario.getBloqueado())) {
                return ResponseEntity.status(HttpStatus.FORBIDDEN)
                        .body(Map.of("error", "Se superó el límite de intentos. Cuenta bloqueada. Revisa tu correo."));
            }
            
            int restantes = 5 - usuario.getIntentosFallidos();
            return ResponseEntity.status(HttpStatus.UNAUTHORIZED)
                    .body(Map.of("error", "Credenciales incorrectas. Te quedan " + restantes + " intentos."));
        }

        // Inicio exitoso: Reseteo de contadores a 0
        seguridadService.resetearIntentos(usuario.getUsername());

        String token = jwtService.generateToken(usuario.getUsername(), usuario.getRol().name());
        return ResponseEntity.ok(new LoginResponseDTO(usuario.getIdUsuario(), token, usuario.getRol().name(), usuario.getUsername()));
    }

    @PostMapping("/desbloquear")
    public ResponseEntity<?> desbloquearCuenta(@RequestBody DesbloqueoRequestDTO request) {
        boolean exito = seguridadService.verificarCodigoDesbloqueo(request.getUsername(), request.getCodigo());
        if (exito) {
            return ResponseEntity.ok(Map.of("mensaje", "Cuenta desbloqueada con éxito. Ya puedes iniciar sesión."));
        } else {
            return ResponseEntity.status(HttpStatus.BAD_REQUEST).body(Map.of("error", "Código inválido o expirado."));
        }
    }

FLUJO COMPLETO DE EJECUCIÓN
═══════════════════════════════════════════════════════════════════════════════

Intento erróneo 1 al 4:
POST /api/auth/login → passwordEncoder de Spring Security falla → Llama a manejarIntentoFallido() → DB suma +1 a intentos_fallidos → Retorna 401 Unauthorized ("Te quedan X intentos").

Quinto intento erróneo consecutivamente:
POST /api/auth/login → falla password → Llama a manejarIntentoFallido() → Llega a 5 → DB asigna bloqueado = true y genera token aleatorio → Invocación asíncrona a NotificationService → Resend envía correo HTML envuelto → AlertaWebService empuja mensaje por WebSocket canal /queue/alertas-{id} → AuthController refresca memoria del usuario y frena la ejecución retornando un estricto 403 Forbidden.

Fase de Recuperación / Entrada de Segundo Factor:
Usuario copia código de su mail o tabla → Llama POST /api/auth/desbloquear enviando JSON con código → SeguridadService convalida vigencia de tiempo y número → DB limpia campos de bloqueo y resetea intentos fallidos a 0 → Devuelve 200 OK. El usuario ya puede autenticarse de forma tradicional y recibir su JWT.
